# Process Scheduling Simulator in C

This project simulates CPU scheduling algorithms using C.

## Algorithms Implemented

- FCFS
- SJF (Non-Preemptive)
- Round Robin

## How to Run

make
./scheduler

## Input

Edit input/processes.txt
## Outcome of the Project
The project successfully simulates CPU scheduling algorithms and evaluates their performance using waiting time and turnaround time metrics, thereby enhancing practical understanding of operating system scheduling mechanisms and C programming.
